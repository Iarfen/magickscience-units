\doxysubsection{Detailed Description}
Class that allows to work with angles. Each angle sizes 4 bytes. 

angle is a class that allows to work with angles, more in degrees than in radians. The value of the angle is always stored inside the class in degrees, and it can be get in radians with the to\+\_\+radian() function.
