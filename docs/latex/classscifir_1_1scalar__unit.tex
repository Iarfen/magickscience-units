\doxysubsection{Detailed Description}
Class that allows to create scalar units. 

\doxylink{classscifir_1_1scalar__unit}{scalar\+\_\+unit} is a class that allows to have a scalar unit, which is composed of a value and one or more dimensions. All dimensions of the SI system of units are supported, and also all of its abbreviations. The abbreviations can be converted to its derived dimensions, too. \doxylink{classscifir_1_1scalar__unit}{scalar\+\_\+unit} classes have operators with all numeric types, and have operators with themselves. They can be displayed by its dimensions, its derived dimensions, or any custom combination of dimensions. When they change dimensions, like, for example, from meters to kilometers, their numeric value is changed accordingly.
