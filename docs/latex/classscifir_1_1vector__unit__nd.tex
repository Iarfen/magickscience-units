\doxysubsection{Detailed Description}
Class that creates a vector in ND, which means a variable number of dimensions. A \mbox{\hyperlink{classscifir_1_1vector__unit__nd}{vector\+\_\+unit\+\_\+nd}} can change to be in 1D, in 2D or in 3D, but it cannot be in more than one dimension at the same time. The member-\/variables of \mbox{\hyperlink{classscifir_1_1vector__unit__nd}{vector\+\_\+unit\+\_\+nd}} are the value and the vector$<$dimension$>$ of \mbox{\hyperlink{classscifir_1_1scalar__unit}{scalar\+\_\+unit}}, and a vector$<$angle$>$ angles. The number of angles is equal to the number of dimensions vector unit has -\/ 1. In 1D \mbox{\hyperlink{classscifir_1_1vector__unit__nd}{vector\+\_\+unit\+\_\+nd}} doesn\textquotesingle{}t has angles, in 2D \mbox{\hyperlink{classscifir_1_1vector__unit__nd}{vector\+\_\+unit\+\_\+nd}} is in {\bfseries{polar coordniates}} an has 1 angle, and in 3D \mbox{\hyperlink{classscifir_1_1vector__unit__nd}{vector\+\_\+unit\+\_\+nd}} is in {\bfseries{spherical coordinates}} an has 2 angles.
