class  scifir::scalar_unit
 Class that allows to create scalar units, which are composed of a value (as a float) and dimensions. The dimensions can be of any number, and be basic dimensions and/or abbreviations of dimensions. Initialization string example: "1 N". More...
 

Namespaces

 scifir
 The namespace scifir contains all scifir-units, excepting the string literals, which are outside.
