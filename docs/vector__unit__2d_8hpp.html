class  scifir::vector_unit_2d
 Class that creates a vector unit in 2D. The vector is in polar coordinates with a value and dimensions of the scalar_unit, and an angle theta for his direction. Initialization string example: "1 N 20θ". 'θ' is the Unicode Character U+03B8. More...
